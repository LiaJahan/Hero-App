My project Name is "Hero App" which is I build for learning purpose.
It allows users to browse, search, and view details of multiple apps. in this page, Users can install apps, which are saved in localStorage, and manage them on a personalized My Installation page with sorting options. The app features dynamic data visualization using Recharts for app reviews, a custom header and footer, and a user-friendly interface that works seamlessly across desktop, tablets and mobile devices. It also includes a custom 404 error page and loading animations to enhance user experience. It also has customized No Apps Found Page.
